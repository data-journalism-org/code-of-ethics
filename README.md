# 倫理綱領

![SPJ倫理綱領の原文](ethicscode.png)

- [オリジナル文書（PDF）](https://www.spj.org/pdf/ethicscode.pdf)
- [Society of Professional Journalists](https://www.spj.org/)

## 前文

プロフェッショナル・ジャーナリスト協会（SPJ）の会員は、市民が啓発されることが正義の先駆けであり、民主主義の基盤であると信じます。
ジャーナリストの責務は、真実を追求し、出来事や問題を公正かつ包括的に伝えることによって、これらの目的を推進することです。
あらゆるメディアと専門分野で活動する良心的なジャーナリストは、徹底した取材と誠実さをもって公共に奉仕するよう努めます。
職業上の高潔さは、ジャーナリストの信頼性の礎です。
協会の会員は倫理的な行動への献身を共有し、協会の原則と実践基準を明示するために、この綱領を採択します。

## 真実を追求し、伝える

ジャーナリストは、情報の収集、報道、解釈において、誠実、公正かつ勇敢でなければなりません。

ジャーナリストは次のことを行います。

- あらゆる情報源から得た情報の正確性を検証し、不注意による誤りを避けるために細心の注意を払います。意図的な歪曲は決して許されません。
- ニュースの当事者を粘り強く探し出し、不正行為の疑いに対して反論する機会を与えます。
- 可能な限り情報源を明らかにします。市民には、情報源の信頼性について可能な限り多くの情報を得る権利があります。
- 匿名を約束する前に、情報源の動機を必ず問い直します。情報提供と引き換えに交わす約束に付随する条件を明確にし、その約束を守ります。
- 見出し、ニュースの予告、宣伝素材、写真、映像、音声、図表、短い発言の抜粋、引用が、事実を誤って伝えないようにします。過度に単純化したり、出来事を文脈から切り離して強調したりしてはなりません。
- ニュース写真や映像の内容を決して歪曲しません。技術的に見やすくするための画像補正は常に認められます。モンタージュや写真によるイラストには、その旨を明示します。
- 誤解を招く再現や演出されたニュース上の出来事を避けます。記事を伝えるために再現が必要な場合は、その旨を明示します。
- 従来の公開された手法では公共にとって重要な情報を得られない場合を除き、潜入取材など、身分や意図を隠す情報収集の手法を避けます。そのような手法を用いた場合は、記事の中で説明します。
- 決して盗用しません。
- たとえ不人気であっても、人間の経験の多様性と広がりを大胆に伝えます。
- 自らの文化的価値観を省み、それを他者に押し付けないようにします。
- 人種、ジェンダー、年齢、宗教、民族、地域、性的指向、障害、外見、社会的地位による固定観念を避けます。
- 自分が嫌悪する意見であっても、意見が自由に交わされることを支持します。
- 声を上げられない人々に発言の機会を与えます。公式、非公式を問わず、情報源は同じように有効であり得ます。
- 主張とニュース報道を区別します。分析や論評であることを明示し、事実や文脈を誤って伝えないようにします。
- ニュースと広告を区別し、両者の境界を曖昧にする混合形式を避けます。
- 公共の事柄が公開の場で扱われ、政府の記録が閲覧できる状態にあるようにする特別な責務を認識します。

## 被害を最小限に抑える

倫理的なジャーナリストは、情報源、取材対象者、同僚を、尊重されるべき人間として扱います。

ジャーナリストは次のことを行います。

- ニュース報道によって悪影響を受ける可能性のある人々に思いやりを示します。子どもや取材に不慣れな情報源・取材対象者には、特に慎重に接します。
- 悲劇や深い悲しみを経験した人々に取材を求めたり、その人々の写真を使用したりする際には、十分に配慮します。
- 情報の収集と報道が、危害や不快感をもたらす場合があることを認識します。ニュースを追求することは、傲慢な振る舞いを許すものではありません。
- 私人は、公職者や権力、影響力、注目を求める人々よりも、自分に関する情報を管理する大きな権利を持つことを認識します。個人のプライバシーへの侵害が正当化されるのは、それを上回る公共上の必要性がある場合に限られます。
- 良識を示し、扇情的な好奇心に迎合しません。
- 少年の容疑者や性犯罪の被害者を特定する際には慎重を期します。
- 正式に起訴される前に犯罪容疑者の氏名を公表することについて、慎重に判断します。
- 犯罪容疑者が公正な裁判を受ける権利と、市民が情報を得る権利との均衡を図ります。

## 独立して行動する

ジャーナリストは、市民の知る権利以外のいかなる利害に対する義務からも自由でなければなりません。

ジャーナリストは次のことを行います。

- 現実のものであれ、そう見受けられるものであれ、利益相反を避けます。
- 高潔さを損なったり、信頼性を傷つけたりする恐れのある団体との関係や活動から距離を置きます。
- 贈答品、便宜、謝礼、無償の旅行、特別待遇を拒否します。また、ジャーナリストとしての高潔さを損なう場合は、副業、政治活動、公職、地域団体での活動を避けます。
- 避けられない利益相反を開示します。
- 権力を持つ者に責任を負わせるため、警戒を怠らず、勇気をもって行動します。
- 広告主や特別な利害関係者を優遇せず、ニュース報道に影響を及ぼそうとする圧力に抵抗します。
- 見返りや金銭と引き換えに情報を提供しようとする情報源に注意し、ニュースの入札競争を避けます。

## 説明責任を果たす

ジャーナリストは、読者、聴取者、視聴者、そして互いに対して説明責任を負います。

ジャーナリストは次のことを行います。

- ニュース報道について明確に説明し、ジャーナリストの行動をめぐる市民との対話を促します。
- ニュースメディアに対する不満や異議を表明するよう、市民に働きかけます。
- 誤りを認め、速やかに訂正します。
- ジャーナリストやニュースメディアによる非倫理的な行為を明るみに出します。
- 他者に求めるものと同じ高い基準を自らにも適用します。

---

SPJ倫理綱領は、活動する場所や媒体を問わず、何千人ものジャーナリストが自発的に受け入れており、ニュース編集部や教室で倫理的な行動の指針として広く用いられています。この綱領は「規則」ではなく、倫理的な意思決定のための資料として意図されたものです。合衆国憲法修正第1条のもとで法的強制力を持つものではなく、また持ち得ません。

現行版の綱領は、協会の会員による数か月にわたる検討と議論を経て、1996年のSPJ全国大会で採択されました。
シグマ・デルタ・カイの最初の倫理綱領は、1926年に米国新聞編集者協会から借用したものでした。1973年にシグマ・デルタ・カイは独自の綱領を作成し、1984年、1987年、1996年に改訂しました。
